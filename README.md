# wb-competitor-analyst
ИИ-Аналитик конкурентов на Wildberries. В проекте взята выборка по категории на Вайлдберрис - рубашки женские, данные за месяц. Исходя из этих данных продавец может оценить насколько его товар отличается от топ-50 в этой категории, получить рекомендации по улучшению карточки товара, что позволит повысить продажи..
Основной функционал:

Анализ отклонений от медианы рынка:сравнение цены, рейтинга, отзывов и продаж с ТОП-50 товарами конкурентов по выбранной нише.
Предрасчет оборачиваемости; автоматический расчет того, на сколько месяцев хватит текущих остатков на складе при текущих темпах продаж.
Генерация бизнес-рекомендаций (LLM): ИИ выступает в роли аналитика, определяя сильные стороны карточки товара, слабые места (например, риск Out-of-Stock или заморозку капитала) и формируя пошаговый план действий.
Строгий JSON-ответ: нейросеть возвращает строго структурированный JSON, готовый для интеграции во фронтенд или дашборды.
Стек технологий: 1.Python 3.10+ 2.FastAPI + Uvicorn — для создания быстрого и асинхронного API. 3.Pydantic — для валидации входящих данных и схем. 4.Pandas — для обработки датасета data.csv и вычисления рыночных медиан. 5.AsyncOpenAI — для асинхронного взаимодействия с LLM (используется Gemini 2.5 Flash через OpenAI-совместимый API).

Установка и запуск:

Скачать данные репозитория
Установите библиотеки: pip install fastapi uvicorn pandas openai pydantic nest_asyncio
Поместите файл с данными data.csv в корневую папку проекта.
Запустите сервер: откройте файл проекта (work1.py) в Jupyter Notebook.

Тестирование:
Перейдите по адресу http://127.0.0.1:8009/docs,чтобы открыть интерактивную документацию Swagger.
Используйте кнопку "Try it out", введите примерные данные своего товара, можно как далее в примере, можно свои. { "price": 1200, "rating": 4.8, "comments": 500, "sales": 7000, "balance": 1000, "temperature": 0 #чтобы получить максимально точные ответы }
На выходе вы получите ответ в поле Response body.
